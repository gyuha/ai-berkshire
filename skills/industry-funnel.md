# 산업 깔때기 선별: 전체 시장에서 3개사까지의 가치투자 정선 프로세스

$ARGUMENTS 산업/방향에 대해 깔때기식 가치투자 선별을 실행하여, 전체 시장 스캔에서 단계적으로 3개 최종 종목까지 압축한다.

## 적용 상황

산업이나 투자 방향(예: "AI 컴퓨팅", "혁신 신약", "로봇")을 제시하고 다음을 원할 때:
1. 중요한 종목을 하나도 빠뜨리지 않기 위해(A주, 홍콩주, 미국주, 비상장 후보 포함)
2. 통일된 기준으로 "이야기주"와 품질 미달 기업을 걸러내기 위해
3. 진짜 깊이 연구할 가치가 있는 3개 선두 기업에 집중하기 위해
4. 각 단계에 명확한 유지/제거 기준이 있어 사후 검토와 추적이 가능하기 위해

`industry-research`와의 차이:
- `industry-research`는 산업 사슬 구조와 전경에 집중, 구간별 분석
- `industry-funnel`은 개별 종목 선별 깔때기에 집중, 전체 시장에서 3개사까지 단계적 압축

둘은 상호 보완적이다: 먼저 `industry-research`로 산업 사슬 구도를 파악한 후, `industry-funnel`로 종목을 선별한다.

## 깔때기 구조 개요

```
1단계: 전체 시장 스캔     30-60개사   （활성도+상승률+시총 상위 30의 합집합）
        ↓ 가치투자 5개 핵심 지표
2단계: 1차 선별           ≤ 10개사   （5개 지표 전부 통과 + 경제적 해자 ★★★ 이상）
        ↓ 정밀 분석
3단계: 정밀 분석          ≤ 10개사  （각 300-500자 구조화 분석）
        ↓ 최종 선별
4단계: 4대 거장 심층 분석  3개사     （각 800-1200자, 워런 버핏·찰리 멍거·돤융핑·리루 4개 시각）
        ↓
출력: 투자 제언 + 매매 신호 + 포지션 제안
```

각 단계에서 "걸러낸 종목"은 반드시 탈락 이유를 남겨야 한다. 블랙박스 처리 금지.

## 1단계: 전체 시장 스캔 진입

### 1.1 활성 주식 정의 (세 가지 합집합)

**A류 - 거래 활성도**:
- 최근 30일 일평균 거래대금 업종 상위 (A주/홍콩주/미국주 각각 상위 30)

**B류 - 상승률 순위**:
- 최근 30일 상승률 상위 20
- 최근 90일 상승률 상위 20
- 양자 합집합

**C류 - 시가총액 앵커**:
- 업종 내 시총 상위 30 (상승/하락 무관)

최종 스캔 풀 = A ∪ B ∪ C, 예상 30-60개사.

### 1.2 반드시 탐색할 시장

| 시장 | 출처 제안 |
|------|---------|
| A주 (상하이·선전) | 통화순/동방재부 업종 판, 통다신 |
| 홍콩주 | 푸투/통화순 홍콩주, HKEX 업종 분류 |
| 미국주 | NASDAQ/NYSE 업종 ETF 편입 종목, Yahoo Finance |
| 국제 시장 | 일·한·대만·유럽 관련 기업 누락 금지 (특히 반도체, 전자) |
| 비상장 기업 | "미래 IPO 후보" 소절로 별도 정리, 최신 기업가치와 잠재 IPO 시기 명시 |

### 1.3 출력 형식

| 회사명 | 코드 | 시장 | 시총 | 주요 사업 한 줄 | 해당 업종 비중 | 입선 분류(A/B/C) |
|-------|------|-----|------|----------|-----------|----------------|

**핵심 자체 점검**:
- 업종 비중 < 30%인 "간접 관련주"는 신중하게 처리, "순수 종목 아님" 표시
- 중국/아시아 시장은 영문 자료 부족을 이유로 누락하지 말 것
- 소시총 기업은 AI가 선두주자를 선호한다는 이유로 누락하지 말 것

## 2단계: 가치투자 5개 핵심 지표 1차 선별 → ≤ 10개사

1단계의 30-60개사에 대해 5개 핵심 지표를 차례로 적용한다.

### 2.1 5개 핵심 지표

| # | 지표 | 통과 기준 | 완화 조건 | 데이터 출처 |
|---|------|---------|---------|---------|
| 1 | PE 밸류에이션 | 합리적 (역사 구간·동업 비교) | 고성장의 경우 PEG < 1.5까지 완화 가능 | 재무제표+wind/통화순 |
| 2 | ROE | > 15% 또는 최근 3년 개선 추세 | 중후장대 업종 완화 가능 | 재무제표 |
| 3 | 영업현금흐름 | 양수이며 순이익 대비 > 70% | — | 재무제표 |
| 4 | 부채비율 | < 60% | 공익사업/전력은 70%까지 완화 가능 | 재무제표 |
| 5 | 경제적 해자 간이 평가 | ★★★ 이상 | — | 정성 판단 |

**경제적 해자 5가지 유형**:
- 브랜드/가격 결정권
- 전환 비용/사용자 고착성
- 네트워크 효과
- 규모의 경제
- 기술/허가증/자원 진입 장벽

### 2.2 출력 형식

| 회사 | PE | ROE | 현금흐름/순이익 | 부채비율 | 경제적 해자 | 종합 | 유지/제거 | 탈락 이유 |
|------|----|----|-----------|-------|-------|------|------|---------|

**유지 규칙**:
- 5개 지표 전부 통과 → 직접 유지
- 4개 통과 + 1개 근접 → 유지하되 황색 표시
- 4개 미만 → 탈락, 이유 명시

**목표**: ≤ 10개사 유지. 과다 유지(> 12개)면 경제적 해자 기준을 ★★★★로 올려 재선별.

## 3단계: 정밀 분석 (≤ 10개사, 각 300-500자)

1차 선별 통과 기업에 대해 차례로 구조화 분석을 실시한다.

### 3.1 각 기업 분석 템플릿

```
## {회사명}（{코드}）

**한 줄 비즈니스 모델**:
（무엇을 팔고, 누구에게 팔고, 어떻게 수익을 내는가）

**재무 품질**:
- 매출 성장률 / 이익 성장률 / 매출총이익률 / ROE / 현금흐름
- 핵심 변화 (최근 1-2년의 가장 중요한 재무 전환점)

**경제적 해자 깊이**:
- 주요 해자 유형 + 구체적 근거
- 5년 후에도 해자가 유지될지 간략 판단

**주요 리스크 (상위 3개)**:
1.
2.
3.

**밸류에이션 간이 평가**:
- 현재 PE/PS/EV/EBITDA + 역사 구간 위치
- 동업 비교
- 한 줄 결론: 고평가 / 합리적 / 저평가

**최종 3개사 진입 여부?**: 예 / 아니오 (이유)
```

### 3.2 최종 3개사 선정 기준

점수 순위로 상위 3개를 선택하는 것이 아니라, "투자 포트폴리오 상호 보완성"으로 선택한다:
- 최소 1개사 "고확실성 저탄력성" (워런 버핏형)
- 최소 1개사 "중간 확실성 중간 탄력성" (성장형)
- 선택적 1개사 "고탄력성 고위험" (옵션형)

특정 세부 트랙에서 3개의 충분히 좋은 기업을 찾지 못하면, "최종 2개사 + 1개 관찰"로 기재하는 것이 낫다. 억지로 수를 채우지 말 것.

## 4단계: 4대 거장 심층 분석 (3개사, 각 800-1200자)

최종 선정 3개사에 대해 4대 거장 시각의 심층 분석을 실행한다.

### 4.1 돤융핑 시각: 비즈니스 본질

- 이 기업이 어떤 사업을 하는지 한 문장으로 정의
- 좋은 사업인가? 왜?
- 이 기업의 "본분"은 무엇인가? 경영진이 이탈하지 않았는가?
- 비즈니스 모델의 "지속성"은 어디에 있는가?

### 4.2 워런 버핏 시각: 경제적 해자 깊이

- 5가지 해자 유형별 점수 (★1-5), 구체적 근거 열거
- 10년 후에도 해자가 유지되는가?
- 현재 매수 시 "안전마진"은 어디에 있는가?

| 경제적 해자 | 강도 | 구체적 근거 |
|-------|------|---------|
| 브랜드/가격 결정권 | | |
| 전환 비용 | | |
| 네트워크 효과 | | |
| 규모의 경제 | | |
| 기술/허가증 진입 장벽 | | |

### 4.3 찰리 멍거 시각: 리스크와 실패 패턴

- 이 기업이 가장 가능성 있는 실패 방식은 무엇인가? (상위 3개 실패 경로 열거)
- 최악의 시나리오에서 얼마나 가치가 있는가? (간단 밸류에이션)
- 똑똑한 사람이 왜 사지 않는가? (역방향 논증)
- 도덕적/컴플라이언스/경영진 리스크가 있는가?

### 4.4 리루 시각: 문명 수준 트렌드 포지셔닝

- 이 기업의 트랙은 "문명급 패러다임 전환"인가, "단계적 열풍"인가?
- 역사상 가장 유사한 기술 혁명의 유사 사례?
- 10-20년 후 이 기업의 최종 결말?
- 승자독식 구도인가?

### 4.5 종합 추천도

```
추천도: ★★★★☆
포지션 유형: 핵심 / 위성 / 옵션 / 관찰
권장 매수 구간: 현재 가격 / N% 조정 후 / 인내심 있게 대기
권장 보유 비중: 해당 테마 포지션의 X%
핵심 모니터링 지표: (이 기업의 투자 논리가 역전될 신호는 무엇인가)
```

## 5단계: 종합 출력

보고서 말미에 통합:

### 5.1 최종 3개사 포트폴리오 표

| 회사 | 유형 | 추천도 | 권장 비중 | 핵심 논리 | 핵심 리스크 |
|------|------|-------|---------|---------|---------|
| A | 핵심 | ★★★★★ | 50-60% | | |
| B | 위성 | ★★★★☆ | 25-35% | | |
| C | 옵션 | ★★★☆☆ | 5-15% | | |

### 5.2 업종 ETF 대안

개별 종목 선택을 원하지 않을 경우, 관련 ETF 1-3개를 열거한다 (A주/홍콩주/미국주).

### 5.3 업종 전체 위치 판단

- 업종 PE/PB 역사 분위
- 자금 흐름 (북향 자금, ETF 신청·환매, 셀사이드 커버리지 밀도)
- 전체가 "초기/확장/성숙/쇠퇴" 중 어느 단계인지

### 5.4 정보 충분도 자체 평가 (필수 작성)

| 차원 | 등급 | 설명 |
|-----|------|-----|
| 기업 재무 데이터 완전성 | A/B/C | |
| 밸류에이션 데이터 시효성 | A/B/C | |
| 업종 구도 판단 | A/B/C | |
| 경영진 정보 | A/B/C | |

A = 데이터 충분하고 신뢰 가능; B = 일부 누락이 있으나 주요 결론에 영향 없음; C = 누락이 많아 결론에 신중 요망.

### 5.5 업데이트 필요 데이터 포인트

명확히 열거: 어떤 데이터가 추정값인지, 어떤 데이터를 후속 확인해야 하는지, 어느 분기 실적 보고서를 집중 추적해야 하는지.

### 5.6 자료 출처 목록

모든 데이터/결론의 출처 링크를 분류별로 열거 (재무제표, 리서치 보고서, 뉴스, 업종 보고서).

## AI 리서치 편향 자각 (중요)

깔때기 선별 과정에서 AI가 쉽게 빠지는 함정:

| 편향 | 발현 | 대응 |
|------|-----|------|
| 선두 기업 선호 | 대시총 기업은 자료가 많고 분석 분량이 길어 "더 좋아 보임" | 핵심 지표와 경제적 해자로 점수 매기기, 보고서 분량으로 순위 매기지 말 것 |
| 영문 선호 | 미국주 자료가 풍부해 A주·홍콩주가 저평가받기 쉬움 | 반드시 한·영 모두 검색, A/H 기업 누락 금지 |
| 이야기 선호 | 높은 상승률 + 미디어 열기 = 더 좋아 보이는 "AI 컨셉주" | "AI 매출 비중" vs "AI 이야기 비중" 구분, 실질 사업 확인 |
| 현재 선호 | 현재 재무가 좋은 기업이 쉽게 입선되어 전환기 다크호스를 놓칠 수 있음 | 2단계 1차 선별에서 "추세 개선"을 완화 조건으로 허용 |
| 상장사 선호 | 상장 기업만 보면 트랙의 최고 기업을 놓칠 수 있음 | 반드시 "미래 IPO 후보" 열거, 기업가치와 시기 창 표시 |

## 출력 요건

1. **보고서 위치**: `reports/{행업명}-funnel-{YYYYMMDD}.md` (업종 보고서는 reports/ 루트 디렉터리에 저장)
2. **언어**: 중문
3. **스타일**: 직접적, 날카롭고, 군말 없이
4. **데이터**: 모든 데이터에 출처 표기; 추정값에는 "추정" 표시
5. **입장 불선제**: 데이터 제시 → 논리 추론 → 결론 도출
6. **정반 양면**: 각 핵심 판단에 반면 논거 첨부
7. **각 단계 탈락 기록 보존**: 탈락 기업도 이름+이유 남기기

## 데이터 검수 (준출 프로세스)

보고서 작성 후, 데이터 검수를 실행하여 통과해야 발행 가능:

```bash
# Step 1 — 검수 목록 추출 (15% 무작위 샘플링)

  --report <보고서 파일 경로>

# Step 2 — 목록의 각 항목을 신뢰 가능한 정보 출처에서 데이터 확인 (skills/financial-data.md 참조)

# Step 3 — 준출/반려 판정 출력
python3 ~/ai-berkshire/tools/report_audit.py verdict \
  --results '<작성 완료된 JSON>' \
  --report <보고서 파일명>
```

**【준출】** 전부 통과 → 보고서 발행 가능; **【반려】** 불통과 항목 존재 → 수정 후 재심.
