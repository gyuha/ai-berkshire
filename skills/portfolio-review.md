# 포트폴리오 관리: "기업 리서치"에서 "포트폴리오 관리"로

$ARGUMENTS에 대해 투자 포트폴리오 점검 및 최적화를 수행한다.

**지원 입력 형식**:
- 보유 목록, 예시: `腾讯30%, 美团20%, 茅台20%, 英伟达15%, 현금15%`
- 또는: `腾讯 500주 @480홍콩달러, 美团 1000주 @130홍콩달러, ...`
- 또는: `내 포트폴리오`(이미 저장된 포트폴리오 파일 `reports/portfolio-latest.md`가 있는 경우)

> "분산투자는 무지에 대한 보호다. 자신이 무엇을 하는지 안다면, 분산투자는 의미가 없다." —— 워런 버핏
>
> "내 평생 본 진짜 좋은 투자 기회는 손가락으로 다 셀 수 있다." —— 리루

## 설계 철학

기업을 리서치하는 것은 투자의 절반에 불과하다. 나머지 절반은 **포트폴리오 차원의 의사결정**이다:
- 얼마나 살 것인가? (비중)
- 어떤 자금으로 살 것인가? (자금 출처 — 신규 자금인가, 교체매매인가)
- 기존 보유 종목과 충돌하지 않는가? (상관관계)
- 최적 포트폴리오는 어떤 모습인가? (기회비용)

워런 버핏은 주식 하나를 절대 고립적으로 보지 않는다 — 그는 항상 "이것이 내가 할 수 있는 최선인가?"를 생각한다.

## 실행 절차

### 1단계: 보유 종목 파싱

입력에서 현재 보유 종목을 파싱하여 다음 형식으로 표준화한다:

| 종목 | 코드 | 보유량 | 매수단가 | 현재가 | 시가총액 | 비중 | 손익 |
|------|------|--------|---------|-------|---------|------|------|

입력이 비율만 있고 금액이 없다면 비율 기준으로 분석한다.

기존 포트폴리오 파일(`reports/portfolio-latest.md`)이 있는지 확인하고, 있으면 읽어서 업데이트한다.

### 2단계: 최신 데이터 수집

Task 도구로 백그라운드 Agent를 시작하고 WebSearch를 통해 각 보유 종목에 대해 병렬로 수집한다:
1. 현재 주가 및 밸류에이션 지표(PE, PB, 배당수익률)
2. 최근 분기 핵심 재무 변화
3. 최근 중요 이벤트
4. 애널리스트 컨센서스(선행 PE, 목표주가)

각 보유 종목에 대해 `tools/financial_rigor.py verify-valuation`으로 밸류에이션 데이터를 검증한다. 각 종목에 정보 풍부도(A/B/C등급)를 표시하고, C등급 종목의 분석 결론에는 낮은 신뢰도를 표시한다.

### 3단계: 개별 포지션 건강 검진

각 보유 종목에 대해 빠른 건강 검진을 수행한다:

| 종목 | 현재 PE | 매수 논리 변화 여부 | 투자 논문 건강도 | 비중 의견 |
|------|:------:|:----------------:|:--------------:|---------|
| 腾讯 | 18x | 변화 없음 | 8/10 | 적정 |
| 美团 | 25x | 경쟁 심화 | 6/10 | 과도, 축소 검토 |

각 보유 종목에 대해 답한다:
- [ ] **오늘 보유하지 않았다면, 현재 가격에 다시 살 것인가?**
- [ ] **내일 거래할 수 없다면, 5년 보유가 편한가?**
- [ ] **매수 논문이 아직 온전한가?**

**돤융핑**: "10년 보유할 생각이 없다면, 하루도 보유하지 마라."

### 4단계: 포트폴리오 차원 분석

#### 4.1 집중도 분석

| 지표 | 현재값 | 권장 범위 | 판단 |
|------|-------|---------|------|
| 1위 종목 비중 | | <40% | |
| 상위 3종목 비중 | | 50-80% | |
| 총 보유 종목 수 | | 5-15개 | |
| 현금 비중 | | 10-30%(시장 환경에 따라) | |

**리루의 기준**: 3-5개 핵심 포지션, 상위 3개가 80% 이상. **단, 각각을 완전히 이해하고 있어야 한다.**

**워런 버핏의 기준**: 핵심 포지션 10개 이하, 단 위성 포지션은 더 허용.

#### 4.2 상관관계 검토

보유 종목 간 숨겨진 연관성을 파악한다:

| 종목A | 종목B | 상관 유형 | 리스크 |
|-------|-------|---------|------|
| 腾讯 | 快手 | 중국 인터넷 동일 섹터 | 규제 리스크 동조화 |
| 英伟达 | 台积电 | AI 공급망 상하류 | AI Capex 동방향 변동 |
| 美团 | 拼多多 | 중국 소비 동일 섹터 | 거시 소비 동방향 변동 |

**체크리스트**:
- [ ] 동일 테마/섹터에 50% 이상 노출되어 있는가?
- [ ] 동일 국가/통화에 50% 이상 노출되어 있는가?
- [ ] 미중 관계가 악화될 경우 포트폴리오 손실은?
- [ ] 글로벌 경기침체 시 포트폴리오 손실은?

#### 4.3 기회비용 분석

이것이 워런 버핏의 가장 핵심적인 사고방식이다 — **모든 돈은 수익이 가장 높은 곳에 있어야 한다**.

모든 보유 종목을 "예상 연간 수익률" 순으로 정렬한다:

| 순위 | 종목 | 현재 비중 | 예상 연간 수익률 | 확실성 | 예상 수익률×확실성 |
|:----:|------|:-------:|:-----------:|:------:|:--------------:|
| 1 | | | | | |
| 2 | | | | | |
| ... | | | | | |

예상 수익률 추정 방법(`tools/financial_rigor.py three-scenario` 사용):
- **간편 공식**: 예상 연간 ≈ FCF Yield + 예상 성장률(주요 방법)
- **가치형 검증**: 안전마진 회귀 + 이익 성장률 + 배당수익률
- **성장형 검증**: 이익 성장률 × 적정 PE 변화

**핵심 질문**: 순위 최하위 종목의 예상 수익률이 현금(무위험 수익률 ~4%)보다 높은가? 그렇지 않다면, 매도하고 현금으로 전환해야 한다.

#### 4.4 스트레스 테스트

| 시나리오 | 가정 | 포트폴리오 예상 영향 | 최대 낙폭 |
|---------|------|-----------------|---------|
| 글로벌 침체 | 기업 이익 20-30% 감소 | | |
| 미중 갈등 격화 | 중국 관련주 50% 할인 | | |
| 금리 급등 | 10년물 국채 → 6% | | |
| 기술 버블 붕괴 | 기술주 PE 40% 압축 | | |

각 시나리오에 대해 정성적+대략적 평가(각 종목의 섹터 속성 및 과거 밸류에이션 변동 범위 기반):
- 어떤 종목이 가장 큰 충격을 받는가? 대략적인 영향 방향과 규모
- 포트폴리오 전체가 감당할 수 있는가?
- 헤지가 필요한가?

### 5단계: 최적화 제안

#### 5.1 리밸런싱 제안

위 분석을 바탕으로 구체적인 리밸런싱 제안을 제시한다:

| 액션 | 종목 | 현재 비중 | 권장 비중 | 이유 |
|------|------|:-------:|:-------:|------|
| 추가매수 | | | | |
| 축소 | | | | |
| 전량매도 | | | | |
| 신규매수 | | | | |
| 유지 | | | | |

#### 5.2 대체 종목 탐색

포트폴리오에 "현금보다 못한" 포지션이 있거나, 현금 비중이 너무 높다면, 이 스킬 내에서 직접 종목을 추천하지 않고 `/industry-research` 또는 `/investment-checklist`를 활용하여 관심 섹터/기업을 체계적으로 스크리닝할 것을 권장한다.

#### 5.3 현금 관리

| 현재 현금 비중 | 권장 현금 비중 | 이유 |
|:-----------:|:-----------:|------|

**워런 버핏**: 현재 $3,820억 현금 보유, 총자산의 25% 이상 — 좋은 기회를 찾지 못할 때, 현금이 최고의 포지션이다.

### 6단계: 포트폴리오 보고서 출력

#### 보고서 구조

```
1. 포트폴리오 개요(보유 표+파이차트 설명)
2. 개별 포지션 건강 검진(각 종목의 건강 상태)
3. 포트폴리오 분석
   - 집중도: 과도 분산/집중 여부?
   - 상관관계: 숨겨진 연관성 및 리스크 동조화
   - 기회비용: 최하위 종목 보유 가치가 있는가?
   - 스트레스 테스트: 극단적 시나리오에서의 낙폭 추정
4. 리밸런싱 제안(구체적 액션+이유)
5. 다음 점검 시기 및 핵심 모니터링 포인트
```

#### 결론은 반드시 명확히 답해야 한다

1. **포트폴리오 전체 건강도**: 우수 / 양호 / 조정 필요 / 심각한 문제
2. **지금 당장 해야 할 한 가지는?**(X 추가매수 / Y 축소 / 유지)
3. **현재 가장 큰 리스크는?**

### 7단계: 포트폴리오 파일 저장

포트폴리오 정보를 `reports/portfolio-latest.md`에 저장한다. 포함 내용:
- 최신 보유 현황표
- 이번 점검 날짜 및 결론
- 리밸런싱 기록(추가 기재)
- 다음 점검 알림

---

## 핵심 원칙

- **모든 돈에는 기회비용이 있다** — 평범한 주식을 보유하는 비용은 훌륭한 주식을 놓치는 것이다
- **집중은 리스크가 아니다, 무지가 리스크다** — 깊이 이해하는 3개 종목을 보유하는 것이, 얕게 아는 30개보다 안전하다
- **현금은 하나의 포지션이다** — 좋은 기회를 찾지 못할 때 현금 보유는 부끄러운 일이 아니다
- **포트폴리오 차원 > 개별 종목 차원** — 좋은 주식도 잘못된 비중에서는 발목을 잡는다
- **정기 점검하되, 과잉매매는 금물** — 분기마다 한 번 점검으로 충분하다, 매일 들여다보며 조정하지 마라
