# 워런 버핏 가치투자 매수 전 Checklist

$ARGUMENTS에 대해 워런 버핏 가치투자 매수 전 Checklist 분석을 실행한다.

**지원 입력 형식**: 단일 또는 복수 기업, 쉼표/공백으로 구분. 예: `腾讯, 茅台, 英伟达` 또는 `NVDA AAPL MSFT`

## 실행 흐름

### 1단계: 입력 파싱, 분석 대상 기업 전체 식별

$ARGUMENTS에서 모든 기업명/코드를 파싱한다. 각 기업에 대해 확인:
- 기업 전체명, 종목코드, 상장 거래소
- 비상장 기업이면 "비상장"으로 표기하고 간략 설명(간접 투자 경로 존재 여부)을 달아 전체 Checklist를 건너뜀

### 1.5단계: AI 리서치 편향 경고

각 기업에 대해 "정보 풍부도" 빠른 등급 평가(A/B/C)를 수행하고 보고서에 표기:

| 등급 | 판단 기준 | Checklist에 미치는 영향 |
|------|---------|-----------------|
| A급 | 상장 기간 충분, 데이터 풍부 | 정상 실행, 단 "컨센서스 함정" 주의 — 지표가 모두 명확해 보인다고 진짜 확실한 건 아님 |
| B급 | 데이터 제한, 추정 필요 | 추정 지표마다 신뢰도 표기, "좋은 사업" 판단 시 데이터 신뢰도 가중 고려 |
| C급 | 정보 극히 부족 | 억지로 6관 표를 채우지 않음, "데이터 부족으로 판단 불가"를 솔직히 표기, 검증 가능한 핵심 문제에 집중 |

**핵심 원칙**: Checklist의 목표는 **나쁜 선택을 걸러내는 것**이다. C급 기업에서 "데이터 부족"은 "불통과"도 "통과"도 아니다 — AI가 표를 못 채운다는 이유로 부결하지 말고, "회색지대, 1차 정보 보완 필요"로 솔직히 표기해야 한다.

돤융핑이 말했다: "'이해 못 하겠다'는 말에는 두 종류가 있다 — 사업이 너무 복잡해서 진짜 이해 못 하는 경우와, 아직 시간을 쏟지 않은 경우다." AI 리서치의 한계는 '자료 부족'과 '이해 불가'를 쉽게 혼동한다는 점이다.

### 2단계: 병렬 데이터 수집

Task 도구를 사용해 **각 기업별** 독립적인 백그라운드 Agent를 실행해 데이터를 수집한다(모든 기업 동시 병렬 실행). 각 Agent가 수집할 항목:

1. **수익성**: ROE(5~10년 추세), 매출총이익률, 순이익률, 잉여현금흐름(FCF)
2. **밸류에이션 데이터**: 현재 주가, 시가총액, PE(TTM), 선행 PE, PB, 배당수익률
3. **성장 추세**: 최근 3년 매출/이익 성장률
4. **재무 건전성**: 부채 수준, 자본지출 필요액, 현금 보유량, 순현금/순부채
5. **경쟁 구도**: 시장점유율, 주요 경쟁사, 점유율 변화 추세
6. **경제적 해자 증거**: 브랜드/전환비용/네트워크 효과/규모의 경제/기술 장벽의 구체적 증거
7. **경영진 기록**: CEO 이력, 주요 의사결정, 지분, 자본배분 기록
8. **최신 동향**: 최근 6개월 주요 사건(실적, M&A, 규제, 경영진 변동 등)

### 3단계: 기업별 6관 Checklist 실행

상장된 각 기업에 대해 순서대로 6관을 통과한다:

---

#### 1관: 이 사업을 이해할 수 있는가 (능력범위)

반드시 답해야 할 것:
- [ ] 이 회사가 무엇으로 돈을 버는지 한 문장으로 설명할 수 있는가?
- [ ] 10년 후에도 대체로 같은 사업을 하고 있을 가능성이 높은가?
- [ ] 성패를 결정하는 핵심 변수는 무엇인가?
- [ ] 이 산업에 대한 인식이 깊은 연구에서 나온 것인가, 아니면 풍문에서 비롯된 것인가?

**평가 기준** (★1-5):
- ★★★★★: 비즈니스 모델이 극히 단순 명확, 10년 확실성 높음 (예: 茅台 — 술 만들어 팖)
- ★★★★☆: 모델은 명확하나 기술적 진입 장벽이 있어 일정 전문 지식 필요
- ★★★☆☆: 모델은 이해 가능하나 10년 확실성 높지 않고 산업 변화 빠름
- ★★☆☆☆: 사업 부문 복잡하거나 산업 격변 중, 미래 예측 어려움
- ★☆☆☆☆: 능력범위 완전 밖

**절대 부결**: 수익 창출 방식조차 설명 못 하면 바로 "능력범위 밖, 분석 불가"로 표기.

---

#### 2관: 좋은 사업인가 (경제적 특성)

데이터로 말한다. **핵심 지표는 반드시 도구로 정확히 계산한다**:

```bash
python3 ~/ai-berkshire/tools/financial_rigor.py verify-valuation \
  --price {주가} --eps {EPS} --bvps {주당순자산} --fcf-per-share {주당FCF} --dividend {주당배당}
```

| 지표 | 해당 기업 수치 | 참고 기준 | 판단 |
|------|-----------|---------|------|
| ROE (5년 평균) | | >15% 우수, >20% 탁월 | |
| 매출총이익률 | | >40% 가격결정력 시사 | |
| 잉여현금흐름(FCF) | | 지속 양(+), ≈순이익 | |
| 자본지출 강도 | | 경자산 > 중자산 | |
| 부채 수준 | | 유이자부채/순이익 < 3년 | |

**평가 기준** (★1-5):
- ★★★★★: ROE>25%, 고마진, 강한 FCF, 경자산, 저부채 (전항목 충족)
- ★★★★☆: 4항목 충족
- ★★★☆☆: 3항목 충족
- ★★☆☆☆: 2항목 충족 또는 추세 악화
- ★☆☆☆☆: 대부분 미충족 또는 FCF 지속 음(-)

---

#### 3관: 경제적 해자가 충분히 깊은가 (경쟁 우위)

항목별 점검:

| 해자 유형 | 보유 여부 | 구체적 증거 | 넓어지는가 좁아지는가? |
|-----------|---------|---------|--------------|
| 브랜드/가격결정력 | | | |
| 전환비용 | | | |
| 네트워크 효과 | | | |
| 비용/규모 우위 | | | |
| 기술/특허 장벽 | | | |

추가 검증: 경쟁사에 100억을 줘도 이 사업을 복제할 수 있는가?

**평가 기준** (★1-5):
- ★★★★★: 복수의 해자가 겹쳐 있고 계속 넓어지는 중
- ★★★★☆: 강한 해자 최소 1개 이상이고 안정적
- ★★★☆☆: 해자 존재하나 얕거나 추세 불명확
- ★★☆☆☆: 해자가 잠식되는 중
- ★☆☆☆☆: 뚜렷한 해자 없음

---

#### 4관: 경영진을 믿을 수 있는가 (사람의 요소)

| 점검 항목 | 평가 |
|--------|------|
| 정직성 (약속 vs 실행) | |
| 자본배분 능력 (자사주매입/배당/M&A 기록) | |
| 주주이익 지향성 (지분, 보수) | |
| 주인 의식 (창업자 vs 전문경영인) | |
| 기업 지배구조 (특수관계거래, 영업권, 감사) | |
| CEO 없어도 정상 운영 가능한가? | |

**평가 기준** (★1-5):
- ★★★★★: 창업자 경영, 자본배분 탁월, 이해관계 완전 일치
- ★★★★☆: 경영진 우수하나 소소한 결함 있음
- ★★★☆☆: 경영진 합격 수준이나 지배구조 리스크 존재
- ★★☆☆☆: 정직성 또는 지배구조 문제 있음
- ★☆☆☆☆: 심각한 정직성 문제 (→절대 부결)

---

#### 5관: 가격이 충분히 저렴한가 (안전마진)

| 지표 | 수치 | 역사적 분위 | 판단 |
|------|------|---------|------|
| PE (TTM) | | | |
| 선행 PE | | | |
| PB | | | |
| 배당수익률 | | | |
| FCF Yield | | | |

추가 검증 (**반드시 도구로 정확히 계산, 암산 금지**):
```bash
python3 ~/ai-berkshire/tools/financial_rigor.py three-scenario \
  --price {주가} --eps {EPS} --shares {발행주식수억} \
  --growth {낙관} {중립} {비관} --pe {낙관PE} {중립PE} {비관PE} --currency {통화}
```
- 3가지 시나리오 밸류에이션 범위 (도구 출력 결과 사용)
- 판단이 틀렸을 때 현재 가격으로 매수하면 최대 손실은?
- 주가 반토막 나도 추가 매수할 수 있는가?

**평가 기준** (★1-5):
- ★★★★★: 내재가치 대비 50% 이하, 극단적 안전마진
- ★★★★☆: 70% 수준, 양호한 안전마진
- ★★★☆☆: 합리적 밸류에이션, 안전마진 보통
- ★★☆☆☆: 고평가, 안전마진 부족
- ★☆☆☆☆: 심각한 고평가

---

#### 6관: 포지션 크기와 의사결정 규율 (감정 통제)

다음 감정 신호를 점검:
- FOMO 때문에 사고 싶은 건 아닌가?
- 남이 추천해서 사고 싶은 건 아닌가?
- 5년간 거래 정지되어도 괜찮은가?
- 매수 논거를 200자 이내로 쓸 수 있는가?

---

### 4단계: 거울 테스트

각 기업에 대해 거울 테스트 문장을 작성한다:

> "나는 ___원에 ___회사를 매수한다. 이유:
> 1. 이 사업의 본질은 ___이며, 나는 이해한다;
> 2. 경제적 해자는 ___이며, 넓어지고/좁아지고 있다;
> 3. 경영진은 ___, 신뢰할 수 있다/없다;
> 4. 현재 가격은 내재가치의 ___ 수준이며, 안전마진이 충분하다/부족하다;
> 5. 내가 틀려도 하방 리스크가 통제 가능하다/불가능하다. 이유: ___."

**5문장을 완성하지 못하면 = 매수하지 않는다.** "통과" 또는 "미통과"를 명확히 표기.

---

### 5단계: 빠른 부결 체크리스트

각 기업에 대해 항목별 점검. 하나라도 해당되면 즉시 "부결"로 표기:

- [ ] 이 회사가 어떻게 돈을 버는지 설명 불가
- [ ] 3년 연속 잉여현금흐름(FCF) 음(-)이며 개선 조짐 없음
- [ ] 경영진 정직성 문제 있음
- [ ] 경쟁 우위가 비가역적으로 잠식되는 중
- [ ] "다음 사줄 사람이 더 높은 가격에 사줘야" 수익 가능한 구조 (바보 게임)
- [ ] 이 투자가 0이 되는 결과를 감당할 수 없음
- [ ] 매수 이유가 주로 "모두가 사고 있어서" 또는 "최근 많이 올라서"
- [ ] 200자 이내로 매수 이유를 쓸 수 없음

---

### 6단계: 종합 비교표 출력 (복수 기업 시 필수)

복수 기업 분석 시 반드시 비교 종합표를 생성한다:

| 기업 | Checklist 통과? | 능력범위 | 좋은 사업 | 해자 | 경영진 | 안전마진 | 핵심 결론 |
|------|----------------|--------|--------|--------|--------|---------|---------|
| | | ★☆☆☆☆ | ★☆☆☆☆ | ★☆☆☆☆ | ★☆☆☆☆ | ★☆☆☆☆ | |

---

### 7단계: 최종 결론 및 파일 저장

각 기업에 대해 명확한 결론을 낸다 (회피 금지):
- ✅ **Checklist 통과** (X/6관) — 심층 리서치 단계 진입 가능
- ❌ **Checklist 미통과** — 어느 레드라인이 트리거되었는지 설명
- ❓ **회색지대** — 핵심 쟁점이 무엇인지, 투자자가 스스로 판단해야 할 사항이 무엇인지 설명
- N/A — 비상장/매수 불가

완성된 보고서를 `~/워런버핏Checklist-[{회사명} 또는 "복수기업비교"].md`에 저장

## 출력 형식 요구사항

1. 기업별 독립 챕터, 포함 내용: 6관 평가표 + 핵심 데이터표 + 핵심 리스크 (3~5개) + 거울 테스트 + 명확한 결론
2. 복수 기업 시 마지막에 종합 비교표 첨부
3. 모든 평가는 반드시 ★ 기호 사용 (★1-5), 반 별 없음
4. 데이터에 출처 시점 표기 필수, 추정값은 반드시 "추정" 표기
5. 말미에 워런 버핏 명언 인용으로 마무리: "투자의 첫 번째 규칙은 돈을 잃지 않는 것"
6. 문체: 직접적, 날카롭게, 군더더기 없이. 워런 버핏/찰리 멍거/돤융핑 어록을 곳곳에 인용

## 핵심 원칙

- **놓치는 것이 틀리는 것보다 낫다**: Checklist의 목표는 나쁜 선택을 걸러내는 것이지 최고를 찾는 것이 아니다
- **능력범위에 솔직하게**: 이해 못 하면 이해 못 한다고 말하고, 억지로 분석하지 않는다
- **안전마진은 생명선**: 좋은 회사도 비싸게 사면 손해다
- **거울 테스트는 건너뛸 수 없다**: 이유를 설명 못 하면 사지 않는다, 예외 없음
