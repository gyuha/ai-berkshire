# 산업 투자 리서치: 산업 가치사슬 전경 스캔 + 4대 거장 종목 분석 프레임워크

$ARGUMENTS 산업에 대해 체계적인 가치사슬 투자 리서치를 수행한다.

## 연구 목표

하나의 투자 주제/논리 체인에서 출발하여 다음을 완수한다:
1. 투자 논리 체인의 각 고리를 검증한다
2. 완전한 가치사슬 전경도를 작성한다
3. 전 세계 모든 상장 기업을 스캔한다 (A주/홍콩주/미국주/국제)
4. 각 세분 고리의 선두 기업에 대해 4대 거장 프레임워크 분석을 실행한다
5. 산업급 투자 포트폴리오 배분 제안을 산출한다

---

## 1단계: 투자 논리 체인 구축 및 검증

### 1.1 논리 체인 작성
화살표 경로로 "저층 트렌드"에서 "수혜 종목"까지의 인과관계를 표현한다. 예시:
```
저층 트렌드 A
    → 수요 B 발생
        → 병목/필수 수요 C 생성
            → 수혜 가치사슬 D
```

### 1.2 고리별 검증
논리 체인의 각 화살표에 의문을 제기하고 근거를 탐색한다:

| 고리 | 핵심 가정 | 검증 방법 | 데이터 출처 |
|------|---------|---------|---------|
| A→B | | 산업 데이터/예측 검색 | |
| B→C | | 수급 분석 검색 | |
| C→D | | 실제 사례/계약 검색 | |

### 1.3 "이미 발생한 검증 이벤트" 탐색
해당 논리 체인을 뒷받침하는 **이미 계약 체결/실행된 실제 비즈니스 이벤트**(예측이 아닌)를 나열한다. 예: 대기업 구매 계약, 정책 문서, 산업 보고서 등.

---

## 2단계: 가치사슬 전경도 작성

### 2.1 가치사슬 구조 작성
산업을 상류→중류→하류→보조 고리로 분해한다. 예시:
```
상류: 원자재/자원 채굴 → 소재 가공/정제
중류: 핵심 장비 제조 → 시스템 통합/공사 → 신기술 R&D
하류: 운영/서비스 → 최종 고객
보조: 검사/인증 → 유지보수 서비스 → 금융 수단(ETF/신탁)
```

### 2.2 각 고리의 "비즈니스 특성" 식별
각 고리에 다음을 표기한다:

| 고리 | 비즈니스 모델 | 매출총이익률 구간 | 경쟁 구도 | 해자 유형 | 경기 민감도 |
|------|---------|-----------|---------|---------|--------|
| | 자원 판매/장비 판매/서비스 판매/임대 | | 독점/과점/완전경쟁 | 자원/라이선스/기술/규모 | 강/중/약 |

### 2.3 "병목 고리" 표시
가치사슬에서 공급이 가장 타이트하고, 대체가 가장 어렵고, 이익률이 가장 높은 고리를 식별한다 — 이 고리가 최적의 투자 종목이 있는 곳이다.

---

## AI 리서치 편향 자각: 산업 리서치의 특수 함정

산업 리서치에서 AI 데이터 편향은 독특한 방식으로 증폭된다:

**산업급 편향**:
| 편향 유형 | 현상 | 대응 |
|---------|------|------|
| 성숙 산업 선호 | 전통 산업(은행/에너지/소비재)은 자료가 많아 AI 분석이 "더 확실해" 보임 | 확실성은 비즈니스 모델에서 오는 것이지 리서치 수량에서 오지 않는다 |
| 신흥 산업 과소평가 | 신산업(AI 응용/합성생물학 등)은 자료가 적어 AI 분석이 보수적 | "현재 데이터"가 아닌 "엔드게임 사고"로 산업 가치를 판단한다 |
| 선두 기업 선호 | 대기업 자료가 소기업보다 훨씬 많아 AI가 자연스럽게 선두를 추천 | 소기업이 더 나은 위험 대비 수익 비율을 가질 수 있다. AI 분석 분량이 짧다고 무시하지 말 것 |
| 상장 기업 선호 | 상장 기업만 스캔하면 가치사슬의 핵심 비상장 플레이어를 놓침 | 반드시 비상장 기업을 검색하고 "미래 IPO 후보"로 표시할 것 |
| 영문 자료 선호 | AI의 영문 자료 처리 능력이 더 강해 중국/아시아 시장 플레이어를 과소평가할 수 있음 | 반드시 한국어/중문/영문 정보 소스를 동시에 검색할 것 |

**가치사슬 스캔의 반편향 조치**:
1. 각 고리마다 "AI가 쉽게 찾는 기업"만 나열하지 말고, "비주류지만 우수할 수 있는 종목"을 능동적으로 검색한다
2. 정보가 부족한 소형주에 대해 분석 분량이 짧다고 추천도를 낮추지 말 것 — 핵심 질문(비즈니스 본질, 경제적 해자, 경영진)으로 평가하지 보고서 길이로 평가하지 않는다
3. 최종 보고서에 각 기업의 "정보 충분도"(A/B/C급)를 표시하여 독자가 AI 분석의 신뢰도를 알 수 있게 한다

## 3단계: 전 세계 상장 기업 스캔

Task 도구를 사용해 백그라운드 Agent를 실행하여 해당 산업의 모든 상장 기업을 전면 검색한다.

### 검색 목록
- 미국주(NYSE/NASDAQ/NYSE American) 관련 기업
- A주(상하이/선전) 관련 기업
- 홍콩주 관련 기업
- 기타 국제 시장(일본/한국/유럽/호주 등)
- 산업 ETF
- 핵심 비상장 기업(미래 IPO 가능성 있는)

### 각 기업에 대해 수집할 항목
- 기업명(한국어/영어/현지어)
- 종목 코드 및 거래소
- 시가총액(근사값)
- 한 줄 설명(가치사슬 내 포지션 및 역할)
- 순수 종목 여부(순수 핵전력 vs 다각화 기업 내 핵전력 사업)
- 가치사슬 소속 고리

### 출력 형식
가치사슬 고리별로 분류하여 각 고리마다 표 하나씩, 스캔된 모든 기업 포함.
투자 확실성에 따라 계층화:
- **Tier 1**: 대형주, 순수 종목, 산업 선두
- **Tier 2**: 중형주, 순수 또는 높은 비중, 세분화 선두
- **Tier 3**: 소형주, 개발 단계, 고위험 고탄력성
- **Tier 4**: 관련 사업을 보유한 다각화 대기업

---

## 4단계: 각 고리 선두 기업 4대 거장 분석

각 가치사슬 고리의 **Tier 1 및 Tier 2 기업**에 대해 다음 분석을 실행한다 (Tier 3/4 기업은 간략 코멘트만):

### 4.1 비즈니스 본질 (돤융핑)
- 이 기업이 가치사슬에서 무엇을 하는지 한 문장으로 정의
- 수익 구조와 성장률
- 매출총이익률/순이익률 수준 및 트렌드
- 현금흐름 특성
- **추가 질문**: 이것은 좋은 비즈니스인가? 왜?

### 4.2 경제적 해자 (워런 버핏)
5가지 경제적 해자 유형으로 평점 (★1-5):

| 경제적 해자 | 강도 | 근거 |
|--------|------|------|
| 브랜드/가격 결정력 | | |
| 전환 비용 | | |
| 네트워크 효과 | | |
| 규모의 경제 | | |
| 기술/라이선스 장벽 | | |

**추가 질문**: 10년 후에도 경제적 해자가 유효한가?

### 4.3 위험 (찰리 멍거)
- 이 기업은 어떻게 실패할 가능성이 가장 높은가?
- 최악의 시나리오에서 기업 가치는 얼마인가?
- 똑똑한 사람들이 왜 사지 않는가?

### 4.4 경영진 (돤융핑 + 워런 버핏)
- CEO/창업자는 누구인가? 핵심 의사결정 기록
- 지분 비율과 이해관계 일치도
- 간략 평가 (A/B/C급)

### 4.5 밸류에이션 스냅샷
- 현재 PE/PS/EV/EBITDA
- 동일 고리 경쟁사와 비교
- 간략 평가: 고평가/적정/저평가

### 4.6 추천도
★1-5로 표시:
- ★★★★★ = 핵심 포지션 후보
- ★★★★☆ = 위성 포지션 후보
- ★★★☆☆ = 관찰 목록
- ★★☆☆☆ = 고위험 옵션
- ★☆☆☆☆ = 비추천

## 5단계: 산업급 위험 평가 (찰리 멍거 "체크리스트")

### 5.1 시스템적 위험 목록

| 위험 | 확률 | 영향 | 대응 전략 |
|------|------|------|---------|
| 투자 논리 체인의 특정 고리 반증 | | | |
| 대체 기술 등장 | | | |
| 정책/규제 블랙스완 | | | |
| 수요 경기 순환적 하락 | | | |
| 밸류에이션 버블 붕괴 | | | |

### 5.2 역사적 유사 사례
역사상 유사한 가치사슬 투자 주제를 찾아 최종 결말을 분석한다:
- 유사 산업은 무엇인가?
- 최종 승자는 누구인가? (상류/중류/하류?)
- 대부분의 투자자는 수익을 냈는가, 손실을 봤는가?
- 현재 산업에 대한 시사점은 무엇인가?

### 5.3 편향 자가 점검
- 서사 편향: 스토리가 너무 완벽하지 않은가?
- 앵커링 효과: 최근 상승률에 앵커링되어 있지 않은가?
- 군중 심리: "모두가 사고 있어서" 따라가는 것은 아닌가?

## 6단계: 문명 트렌드 판단 (리루 프레임워크)

- 이 산업이 기반하는 저층 트렌드는 "문명급 패러다임 전환"인가, 아니면 "일시적 붐"인가?
- 역사상 가장 유사한 기술 혁명의 유사 사례는 무엇인가?
- 10~20년 후 이 산업의 엔드게임은 무엇인가?
- 가치사슬에서 어느 고리가 "승자 독식"이 될 가능성이 가장 높은가?
- 어느 고리가 가장 파괴될 가능성이 높은가?

## 7단계: 포트폴리오 배분 제안

### 7.1 추천 포트폴리오

|------|---------|------|---------|---------|

## 출력 요건

1. 모든 분석에는 데이터 근거가 있어야 하며, 데이터 출처를 첨부한다
2. 핵심 데이터는 Markdown 표로 표현한다
3. 가치사슬 전경도는 코드 블록의 텍스트 차트로 표현한다
4. 각 고리마다 최소 2~3개 선두 기업을 분석한다
5. 전 세계 기업 스캔은 가능한 한 완전하게 한다 (A주/홍콩주/미국주/국제)
6. 최종 완성 보고서를 `~/[산업명]산업가치사슬투자연구보고서.md`에 저장한다
7. 결론은 명확하게, 구체적인 종목·비중·가격 구간 제안을 제시한다
8. 각 분석 모듈 끝에 해당 거장의 "추가 질문"을 첨부한다

## 데이터 샘플링 검사 (출고 절차)

보고서 작성 후 데이터 샘플링 검사를 실행하고 통과하면 발행한다:

```bash
# Step 1 — 샘플링 체크리스트 추출 (15% 무작위 샘플링)
python3 ~/ai-berkshire/tools/report_audit.py extract \
  --report <보고서 파일 경로>

# Step 2 — 체크리스트 각 항목을 신뢰할 수 있는 출처에서 데이터 확인 (skills/financial-data.md 참조)

# Step 3 — 출고/반려 판정 출력
python3 ~/ai-berkshire/tools/report_audit.py verdict \
  --results '<작성된 JSON>' \
  --report <보고서 파일명>
```

**【출고】** 전부 통과 → 보고서 발행 가능; **【반려】** 불통과 항목 있음 → 수정 후 재심사.
