# 공급망 병목 헌터: AI 기반 글로벌 산업망 병목 차익거래

$ARGUMENTS 슈퍼트렌드에 대해 공급망 병목 스캔과 차익거래 기회 발굴을 실행한다.

## 핵심 이념

"AI가 어떤 주식을 추천하는가"를 묻지 말고, "이 트렌드가 계속 확장된다면 어느 고리가 먼저 부족해지는가"를 물어라.

전통적 투자 리서치는 선도기업과 알려진 섹터만 바라본다. 이 시스템은 반대로 접근한다: **물리적 공급망의 목구멍 위치에서 출발해, 아무도 주목하지 않지만 한 번 품절되면 산업 전체가 멈춰 기다려야 하는 기업을 찾는다**.

초과 수익의 원천: 1층 병목(GPU, HBM, 전력)은 이미 충분히 가격에 반영됐다. 진짜 알파는 **2층, 3층**에 있다 — 광모듈, 레이저, InP 기판, SOI 웨이퍼, 에피택시 장비, 웨이퍼 레벨 테스트, IC 기판, 특수 유리섬유 등.

## 1단계: 슈퍼트렌드 확인

### 1.1 트렌드 필터링 기준

소규모 테마에서 환상을 찾지 말고, 아래 조건을 모두 충족하는 슈퍼트렌드만 추적한다:

| 기준 | 요건 | 검증 방법 |
|------|------|---------|
| 지속성 | 최소 3~5년 확실한 성장 | 업계 예측, 설비투자 계획 검색 |
| 물리성 | 실제 하드웨어/소재/설비 건설 필요 | "소프트웨어 업그레이드"와 "물리적 확장" 구분 |
| 규모성 | 글로벌 설비투자 >500억 달러/년 | 선도 기업 capex 가이던스 검색 |
| 가속성 | 수요 증가속도 > 공급 증설속도 | 수요성장률 vs 생산능력 확장 계획 비교 |

### 1.2 현재 추적 중인 슈퍼트렌드 목록

매 실행 시 업데이트, 초기 목록:

1. **AI 인프라 구축** — 데이터센터, GPU 클러스터, 네트워크 인터커넥트, 전력
2. **에너지 전환** — 원전 재가동, 송전망 업그레이드, 에너지 저장
3. **국방 현대화** — 서방 국방비 상승 사이클, 공급망 재편
4. **반도체 재산업화** — 미·유럽·일본 보조금 공장 건설, 장비/소재 병목
5. **우주 경제** — 위성 인터넷, 발사 빈도 급증

사용자가 특정 트렌드(예: "AI 인프라")를 지정한 경우 해당 트렌드에만 집중한다.

### 1.3 트렌드 검증 출력

```
트렌드명:
핵심 동인: (한 문장)
이미 발생한 검증 이벤트 (최소 3개):
  1. [날짜] [이벤트] [출처]
  2.
  3.
설비투자 규모: 글로벌 약 $XX 억/년, 증가율 YY%
수급 갭 판단: 수요 증가속도 > 공급 증설속도? 예/아니오/불확실
트렌드 확인: ✅ 추적 가능 / ❌ 증거 불충분, 잠시 추적 중단
```

## 2단계: 공급망 물리적 분해

### 2.1 계층 분해 프레임워크

**개념 단계에 머물지 말고, 물리적 실체까지 분해한다**.

```
Layer 0(최종): 최종 제품/서비스
    │
Layer 1(핵심 부품): 시장이 충분히 주목하는 핵심 하드웨어
    │                 ⬆ 가격 충분히 반영, 알파 제한
    │─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
    │                 ⬇ 관심도 낮음, 알파 집중 구간
    │
Layer 2(서브 부품/소재): 핵심 부품을 뒷받침하는 부품과 소재
    │
Layer 3(상류 장비/원료): 서브 부품 제조에 필요한 장비와 원자재
    │
Layer 4(기반 인프라): 전력, 냉각, 토지, 인재, 인증
```

### 2.2 AI 인프라를 예시로 한 분해 템플릿

```
Layer 0: AI 모델 훈련/추론 서비스
Layer 1: GPU/가속기, HBM 메모리, 서버, 데이터센터
Layer 2(중점 스캔 구간):
  ├─ 네트워크 인터커넥트: 광모듈, 광섬유, 스위치 칩, 구리 케이블
  ├─ 광통신 핵심: 레이저(EML/VCSEL/CW), 변조기, 광검출기
  ├─ 반도체 소재: InP 기판, GaAs 기판, SOI 웨이퍼, SiC 기판
  ├─ 첨단 패키징: CoWoS 기판, HBM TSV, ABF 기판 필름
  ├─ PCB/기판: 고주파 고속 PCB, IC 기판, 특수 유리섬유포
  ├─ 테스트: 웨이퍼 레벨 테스트(Probe Card), 번인 테스트, ATE
  ├─ 방열/냉각: 액냉 시스템, CDU, 침수형 냉각액
  └─ 전력 연결: 부스바, UPS, 배전반, 변압기
Layer 3:
  ├─ 에피택시 장비: MOCVD, MBE
  ├─ 리소그래피/식각: 특수 파장 리소그래피, InP 식각
  ├─ 원자재: 고순도 금속(인듐, 갈륨, 게르마늄), 특수 가스, 타겟재
  └─ 인증/표준: MSA 표준, Telcordia 인증
Layer 4:
  ├─ 전력: 원전, 천연가스 발전, 송변전
  ├─ 냉각수/방열 기반 인프라
  └─ 데이터센터 토지/허가
```

## 3단계: 병목 식별 — "목구멍" 찾기

### 3.1 병목 판정 6가지 기준

Layer 2~3의 각 고리에 대해 순서대로 평가한다:

| # | 기준 | 질문 | 점수 |
|---|------|------|------|
| 1 | **공급 집중도** | 글로벌 공급업체 ≤3개? | 🔴 ≤2개 / 🟡 3~5개 / 🟢 >5개 |
| 2 | **증설 주기** | 신규 생산능력에 얼마나 걸리나? | 🔴 >2년 / 🟡 1~2년 / 🟢 <1년 |
| 3 | **대체 난이도** | 다른 기술/소재로 대체 가능? | 🔴 대체 불가 / 🟡 부분 대체 / 🟢 쉽게 대체 |
| 4 | **가동률** | 현재 생산능력 가동률? | 🔴 >90% / 🟡 70~90% / 🟢 <70% |
| 5 | **수요 증가속도** | 하류 수요 증가속도? | 🔴 >50%/년 / 🟡 20~50% / 🟢 <20% |
| 6 | **고객 검증 주기** | 신규 공급업체 진입 검증에 얼마나 걸리나? | 🔴 >1년 / 🟡 6~12개월 / 🟢 <6개월 |

**병목 등급**:
- 🔴🔴🔴 ≥4개 → **S급 병목** (단일 장애점 수준, 최우선순위)
- 🔴🔴 3개 → **A급 병목** (심각한 제약)
- 🔴 1~2개 → **B급 병목** (압박 있지만 통제 가능)
- 🔴 없음 → 병목 아님, 건너뜀

### 3.2 병목 맵 출력

```
공급망 병목 맵 — {트렌드명}
업데이트일: YYYY-MM-DD

S급 병목(단일 장애점):
  1. [고리명] — [이유 한 문장] — 공급업체: [기업 목록]
  2.

A급 병목(심각한 제약):
  1.
  2.

B급 병목(압박 있음):
  1.
  2.

최근 변화 (vs 직전 스캔):
  - [신규/승급/강등/해제] [고리명] — [이유]
```

## 4단계: 기업 필터링 — 병목에서 투자 대상으로

### 4.1 S급 및 A급 병목마다 관련 상장 기업 전부 발굴

검색 방식:
- WebSearch "{병목 고리} supplier listed company"
- WebSearch "{병목 고리} manufacturer stock"
- WebSearch "{병목 제품} market share company"

### 4.2 초기 필터링 기준 (빠른 스크리닝)

| 기준 | 요건 | 이유 |
|------|------|------|
| 상장 여부 | 상장 완료(A주/홍콩/미국/일본/대만/유럽) | 거래 가능 |
| 병목 사업 비중 | 병목 고리 매출 >30% | 순도 |
| 시가총액 | 우선 <100억 달러 | 대형주는 이미 충분히 가격에 반영 |
| 유동성 | 일평균 거래금액 >100만 달러 | 진출입 가능 |

### 4.2.1 밸류에이션 점검 (필수 실행, 건너뜀 불가)

**병목이 실재함 ≠ 투자 기회.** 모든 기업에 대해 PS, PE를 계산하고 보고서에 기재해야 한다. 아래 조합 조건으로 밸류에이션 과열 여부를 판단한다:

#### 밸류에이션 적신호 (하나라도 해당 → 신호 강도 최대 ★★, "⚠️ 밸류에이션 과열" 표기)

1. **시가총액 > TAM의 20%**: 기업 시가총액이 도달 가능한 시장 규모의 20%를 이미 초과 — 성장 기대치가 과도하게 내재화됨
2. **PS > 30x 이고 매출 성장률 < 100%**: 고밸류에이션이지만 증가속도가 불충분. 증가율>100% 기업은 PS 기준 면제 가능하지만 "⚠️ 고밸류에이션은 높은 성장 지속 검증 필요" 표기
3. **시가총액 > 5년 낙관적 매출 예측의 10배**: 가장 낙관적인 가정이 모두 실현되어도 현재 가격은 과도함
4. **유상증자 후 60일 이내 주가 2배**: 감정 주도 특성 명확, 신호 강도 한 단계 강등

#### 밸류에이션 황색 신호 (추가 설명 없으면 강등)

1. **적자 + PS > 15x**: ★★★ 진입 허용하지만 수익화 경로와 일정 반드시 설명
2. **PS가 동업 수익 기업의 5배 이상**: 프리미엄 원천 설명 필수 (시장점유율, 성장속도 차이, 진입장벽 차이)
3. **PE > 80x**: PEG 계산하고 성장속도가 이를 뒷받침하는지 설명 필요

#### 밸류에이션 녹색 신호 (가점 항목)

- PS < 10x 이고 매출 성장 중 → 신호 강도 한 단계 상향 가능
- PE < 30x 이고 경제적 해자 있음 → "밸류에이션에 안전마진 존재" 표기

#### 밸류에이션 합리성 검증 (필수)

모든 투자 대상에 대해 답한다: "현재 시가총액으로 매입, 가장 낙관적 시나리오가 모두 실현된다고 가정, 10년 후 25x PE로 매도하면 연환산 수익률은 얼마인가?" 연환산 수익률 <10% → "현재 가격에 안전마진 없음" 표기.

**주의**: 밸류에이션 점검의 목적은 "PS 100배인 적자 기업" 추천 같은 명백한 오류를 방지하는 것이지, 모든 고밸류에이션 초기 기업을 배제하는 것이 아니다. 핵심은 성장속도, TAM, 경쟁 구도가 현재 밸류에이션을 뒷받침할 수 있는지이며, 구체적 분석이 필요하지 일도양단식 판단은 안 된다.

### 4.3 심층 필터링 차원

초기 필터링을 통과한 기업에 대해 개별 평가한다:

```
## {회사명}({코드})

**병목 포지셔닝**:
- 공급망에서의 구체적 위치
- 시장점유율: 글로벌 X위, 비중 XX%
- 고객 목록 (알려진 것)

**생산능력 및 확장**:
- 현재 생산능력 / 가동률
- 증설 계획 / 일정
- 증설 필요 자금 vs 현재 현금

**재무 스냅샷**:
- 시가총액 / 매출 / 이익 / 성장률
- 병목 사업 매출 비중
- 매출총이익률 추세 (병목이 심할수록 매출총이익률 상승해야 함)

**리스크 점검 체크리스트**:
- [ ] 대체 기술 리스크: 우회 가능한가?
- [ ] 희석 리스크: 대규모 유상증자/전환사채 있나?
- [ ] 지정학적 리스크: 민감 지역에 있거나 수출 통제 대상인가?
- [ ] 경영진 리스크: 불량 기록 있나?
- [ ] 고객 집중 리스크: 단일 고객에 과도하게 의존하나?
- [ ] 밸류에이션 과열: 현재 밸류에이션이 이미 3년 성장을 반영하나?

**병목 지속성 판단**:
- 이 병목은 언제 해소될 것인가?
- 해소된 후 이 기업에게 남는 것은 무엇인가?
- 일회성인가 지속적인가?
```

## 5단계: 교차 검증 — 하나의 이야기만 듣지 않기

### 5.1 정방향 검증

| 검증 항목 | 질문 | 검색 방식 |
|--------|------|---------|
| 고객 검증 | 선도 고객이 이미 계약/도입했나? | 기업 공시, 고객 재무 보고서 언급 검색 |
| 매출 검증 | 병목이 이미 매출 성장에 반영됐나? | 최근 2~3분기 재무 보고서 검색 |
| 가격 검증 | 제품이 인상 중인가? | 업계 시세, 애널리스트 보고서 검색 |
| 생산능력 검증 | 생산능력이 실제로 타이트한가? | 납기 데이터, 고객 불만 검색 |
| 자본 검증 | 증설 설비투자가 있나? | 기업 capex 가이던스 검색 |

### 5.2 역방향 검증 (찰리 멍거식 부정)

| 역방향 질문 | 의미 |
|---------|------|
| 똑똑한 사람들이 왜 이 주식을 안 사는가? | 알려진 약세론 찾기 |
| 이 병목을 우회할 수 있나? 대체 경로가 있나? | 기술 경로 리스크 |
| 중국/다른 플레이어가 빠르게 생산능력을 복제할 수 있나? | 공급 충격 리스크 |
| 최종 수요가 50% 둔화되면 이 기업은 어떻게 되나? | 하방 민감도 |
| 경영진이 과거 고점에서 유상증자로 희석한 적 있나? | 경영진 신뢰도 |
| 현재 밸류에이션에 내포된 성장 가정은 무엇인가? | 밸류에이션 합리성 |

### 5.3 신호 교차 검증

## 매시간 스캔 모드 (정시 작업 사용)

매시간 한 번 실행하며, "내용이 있을 때만 보고서를 출력"하는 방식 채택:

### 스캔 흐름 (매시간)

1. **뉴스 스캔**: 지난 1~2시간의 공급망 관련 뉴스 검색
   - 키워드: supply chain bottleneck, shortage, capacity constraint, allocation, lead time, sole source, 瓶颈, 缺货, 产能, 涨价
   - 범위: 영문 + 중문 소스
2. **시장 신호**: 추적 중인 기업의 주가 변화 점검(특히 이상 변동 >5%)
3. **재무 보고서/공시**: 병목 관련 기업의 재무 보고서 또는 중요 공시 여부 점검
4. **밸류에이션 기회**: 관찰 목록 기업이 대형 하락 등으로 매수 구간에 진입했는지 점검
5. **보고서 출력 여부 판단**:
   - 새로운 병목 신호, 명확한 투자 대상 기회, 중요 상태 변화 → **보고서 출력**
   - 새로운 발견 없음 → **보고서 미출력**, 로그에만 "본 라운드 신규 신호 없음" 기록

## 핵심 원칙 (최우선순위)

1. **AI에게 주식 추천을 시키지 말고, AI에게 공급망을 분해하게 하라** — 질문이 답보다 중요하다
2. **물리 우선** — 실제 물리적 제품/소재/장비가 필요한 고리만 관심
3. **2층, 3층** — 이미 충분히 가격에 반영된 선도기업 추종 금지
4. **교차 검증** — 모든 결론은 최소 2개의 독립 정보 소스
5. **불확실성에 솔직하게** — 데이터를 못 찾으면 "데이터 불충분" 기재, 추측으로 채우지 않음
6. **병목에는 유효기간이 있다** — 모든 병목은 결국 해소된다, 핵심은 시간 창을 판단하는 것
7. **소형주 ≠ 좋은 기회** — 소형주도 나쁜 기업일 수 있다, 재무 품질 관문을 반드시 통과해야 함
8. **병목이 실재함 ≠ 투자 기회** — 한 기업이 가장 빡빡한 병목에 앉아 있어도 PS>30x이거나 여전히 적자면 현재 가격은 매수 시점이 아니다. **밸류에이션은 하드 허들이며, 병목 순도, 신호 강도, 서사의 매력으로 덮어씌울 수 없다.** 오른 병목주를 놓치는 것이 낫지, PS 100배인 적자 기업을 매수해서는 안 된다
9. **CLAUDE.md 객관성 원칙 준수** — 강세를 전제하지 않고, 데이터 먼저 결론 나중
