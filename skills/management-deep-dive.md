# 경영진 심층 연구: 주식 매수는 곧 사람에 대한 투자

$ARGUMENTS에 대한 경영진 심층 연구를 수행한다.

**지원 입력 형식**: `회사명` 또는 `인물명 회사명`, 예: `美团`、`王兴 美团`、`黄仁勋 英伟达`

> "주식 매수는 곧 사람에 대한 투자다. 신뢰할 수 있는 사람을 찾아 장기 보유하라." — 돤융핑
>
> "경영진을 평가할 때는 아무도 보지 않을 때 무엇을 하는지를 보라." — 워런 버핏

## 설계 철학

대부분의 투자 분석은 경영진 평가를 표면에 머문다: 이력서, 지분 비율, 보수. 그러나 워런 버핏은 많은 시간을 **경영진과 식사하며 대화하는 데** 쏟고, 리루는 **투자의 본질은 사람에 대한 투자**라고 말하며, 돤융핑은 **주식 매수는 곧 사람에 대한 투자**라고 말한다.

이 Skill은 `/investment-research` 5단계 경영진 평가의 **심화 버전**이다. 표준 투자 리서치에서 경영진 평점이 불확실(★★★ 이하)하거나, 경영진이 핵심 투자 논거일 때 이 Skill로 심층 연구를 수행한다.

AI는 경영진과 직접 식사할 수 없지만, 공개 정보를 통해 다음을 수행할 수 있다:
- **경영진의 말과 행동이 일치하는지 추적**(약속 vs. 이행)
- **모든 중요한 자본 배분 결정의 수익률 분석**
- **어려운 시기의 결정을 통해 품성 추론**
- **직원/파트너/고객의 피드백을 통한 측면 검증**

## 실행 흐름

### 1단계: 핵심 경영진 파악 및 병렬 데이터 수집 시작

WebSearch로 다음 핵심 인물을 확인한다:

| 역할 | 성명 | 재임 기간 | 배경 | 지분/옵션 |
|------|------|----------|------|----------|
| CEO/이사회 의장 | | | | |
| CFO | | | | |
| 창업자(현직 아닐 경우) | | | | |
| 실질 지배자(CEO와 다를 경우) | | | | |
| 기타 핵심 임원 | | | | |

**주의**: "누가 결정을 내리는가"와 "누구의 이름이 직함에 있는가"를 구분한다. 일부 회사는 창업자가 물러났어도 여전히 핵심 인물인 경우가 있다(예: 黄峥과 拼多多).

핵심 인물을 확인한 후, Task 도구로 복수의 백그라운드 Agent를 시작해 다음 데이터를 **병렬** 수집한다:
1. Agent 1: CEO 공개 발언 및 예측 기록(주주 서한, 실적 발표 콜, 인터뷰, 소셜미디어)
2. Agent 2: 자본 배분 결정 기록(인수합병, 자사주 매입, 배당, 신사업 투자)
3. Agent 3: 지배구조 및 보수(지분 구조, 특수관계자 거래, 임원 보수)
4. Agent 4: 측면 검증 정보(직원 평가, 고객 피드백, 업계 평판)

### 2단계: CEO 능력범위 평가

#### 2.1 전략적 통찰력

CEO의 지난 5년간 공개 발언(주주 서한, 실적 발표 콜, 인터뷰, 소셜미디어)을 검색해 다음 사안에 대한 판단을 추출한다:

| 시점 | CEO의 판단/예측 | 실제 결과 | 정확도 |
|------|--------------|---------|:------:|
| | "우리는 X시장이 ..."라고 본다 | X시장 실제 ... | ✅/❌ |
| | "향후 3년 우리의 중점은 ..." | 실제 실행 ... | ✅/❌ |

**핵심 질문**:
- CEO가 시장보다 앞선 올바른 판단을 내린 적이 있는가?
- 모두가 낙관적일 때 냉정함을 유지한 적이 있는가?
- CEO의 산업 트렌드 이해는 시장을 추종하는가, 독립적으로 사고하는가?

#### 2.2 실행 능력

| 항목 | 평가 | 근거 |
|------|------|------|
| 전략→실행 | 말한 것을 실행했는가? | |
| 조직 역량 | 인재를 유치하고 유지할 수 있는가? | |
| 위기 대처 | 어려움에 처했을 때 어떻게 대응했는가? | |
| 수정 속도 | 실수 후 빠르게 수정했는가? | |

### 3단계: 성실성 평가(가장 중요)

**워런 버핏**: "우리는 세 가지 자질을 찾는다: 성실성, 지성, 에너지. 첫 번째가 없으면 나머지 둘이 당신을 해친다."

#### 3.1 약속 vs. 이행 추적

지난 3년간의 실적 발표 콜, 주주 서한, 공개 인터뷰에서 경영진이 한 **구체적인 약속**을 추출한다:

| # | 시점 | 약속 내용 | 약속 장소 | 이행 여부 | 평가 |
|---|------|---------|---------|---------|------|
| 1 | | "2025년까지 X사업 흑자 전환할 것" | 2024년 실적 발표 콜 | | ✅/⚠️/❌ |
| 2 | | "$X억 자사주 매입 계획" | 2024년 주주 서한 | | ✅/⚠️/❌ |

**이행률 통계**:

| 약속 이행률 | 평가 |
|:---------:|------|
| >80% | 우수 — 말한 것을 실행함 |
| 60-80% | 합격 — 방향은 맞으나 실행에 편차 |
| 40-60% | 우려 — 약속은 많고 이행은 부족 |
| <40% | 심각한 문제 — 신뢰 불가 |

#### 3.2 어려운 시기의 대처

회사 역사상 겪은 주요 위기/어려움(주가 급락, 실적 미스, 규제 충격, 경쟁 심화)을 검색해 경영진의 대응을 분석한다:

| 위기 사건 | 시점 | 경영진 반응 | 사후 평가 |
|---------|------|-----------|-------------|

**주목할 점**:
- 능동적으로 소통했는가, 아니면 회피했는가?
- 내부 요인을 인정했는가, 외부 탓으로 돌렸는가?
- 어렵지만 올바른 결정을 내렸는가, 단기 시장 영합을 선택했는가?

#### 3.3 이해관계자에 대한 태도

| 이해관계자 | 경영진 태도 | 근거 | 평가 |
|-----------|-----------|------|------|
| 주주 | 존중/무시/이용 | | |
| 직원 | 우대/착취/방치 | | |
| 고객/이용자 | 고객 중심/단기 수탈 | | |
| 파트너/공급업체 | 공정 협력/극단적 단가 압박 | | |
| 규제당국/사회 | 준법 협력/경계선 활용 | | |

**리루**: "이해관계자에 대한 태도가 기업의 장기 생명력을 결정한다. 단기 착취는 효율을 높일 수 있지만 장기적으로 생태계를 훼손한다."

### 4단계: 자본 배분 능력

이것이 워런 버핏이 경영진에서 가장 중요하게 보는 능력이다 — **1원을 벌었을 때, 경영진이 그것을 얼마로 불릴 수 있는가?**

#### 4.1 자본 배분 결정 기록

회사의 지난 5년간 주요 자본 배분 결정을 검색해 건별로 평가한다:

**인수합병 기록**:

| 시점 | 인수 대상 | 금액 | 전략 논리 | 사후 수익률 | 평점(1-5) |
|------|---------|------|---------|---------|:---------:|

**자사주 매입 기록**:

`tools/financial_rigor.py verify-valuation`을 사용해 매입 당시 및 현재의 PE 등 밸류에이션 지표를 검증한다.

| 시점 | 매입 금액 | 평균 매입가 | 당시 PE | 사후 평가 | 평점(1-5) |
|------|---------|-----------|:------:|---------|:---------:|

**배당 기록**:

| 연도 | 배당 금액 | 배당성향 | 동기간 FCF | 지속 가능 여부 |
|------|---------|:------:|---------|:---------:|

**신사업 투자**:

| 시점 | 투자 분야 | 누적 투입 | 현황 | 수익률 평가 | 평점(1-5) |
|------|---------|---------|---------|---------|:---------:|

#### 4.2 자본 배분 평점

| 항목 | 평점(1-5) | 설명 |
|------|:---------:|------|
| 인수합병 규율 | | 합리적 가격에 인수했는가? 인수 후 통합은 어떠했는가? |
| 자사주 매입 타이밍 | | 저평가 시 매입, 고평가 시 중단했는가? |
| 배당 합리성 | | 배당성향이 FCF와 일치하는가? |
| 신사업 투자 | | 성공률은? 손절 규율은? |
| 현금 관리 | | 현금 보유가 적정한가? 과도하게 쌓아두지는 않는가? |
| **종합 평점** | | |

**워런 버핏 기준**: 이상적인 경영진은 좋은 기회가 있을 때 과감히 투자하고, 좋은 기회가 없을 때 적극적으로 자사주 매입/배당을 실시하며, 고가 인수합병은 절대 하지 않는다.

### 7단계: CEO 부재 시나리오 분석

**워런 버핏**: "좋은 회사는 바보도 경영할 수 있어야 한다 — 언젠가는 반드시 바보가 경영하게 될 것이기 때문이다."

| 질문 | 답변 |

### 8단계: 경영진 평가 보고서 작성

#### 보고서 구조

```
1. 핵심 인물 개요(표)
2. 성실성 평가
   - 약속 이행률
   - 어려운 시기의 대처
   - 이해관계자에 대한 태도
3. 역량 평가
   - 전략적 통찰력(예측 정확도)
   - 실행 능력
   - 자본 배분 기록
4. 지배구조
   - 지분 구조 위험
   - 보수 적정성
   - 특수관계자 거래
5. 측면 검증
   - 직원 시각
   - 고객/파트너 시각
6. 종합 평점 및 결론
```

#### 종합 평점

| 항목 | 가중치 | 평점(1-5) | 가중 점수 |
|------|:----:|:---------:|:----:|
| 성실성 | 35% | | |
| 전략 및 실행 능력 | 25% | | |
| 자본 배분 능력 | 25% | | |
| 지배구조 | 15% | | |
| **종합 평점** | 100% | | |

#### 돤융핑의 "사람 매수" 기준

> 다음 세 가지 질문에 답하라:
> 1. **이 사람은 성실한가?**(정직하며 주주 이익을 해치지 않는가)
> 2. **이 사람은 유능한가?**(전략적 통찰력 + 실행력 + 자본 배분)
> 3. **이 사람에게 10년간 돈을 맡길 수 있는가?**
>
> 셋 모두 "예" = ★★★★★(5점)
> 앞의 둘이 "예" = ★★★★(4점)
> 첫 번째만 "예" = ★★★(3점)
> 첫 번째가 "예"가 아님 = ★(1점, 투자 불가)

## 핵심 원칙

- **성실성은 거부권 항목이다** — 능력 부족은 학습으로 보완 가능하지만, 품성 문제는 수정이 불가능하다
- **말이 아닌 행동을 보라** — 경영진이 무엇을 말했는지는 중요하지 않다, 무엇을 했는지가 중요하다
- **어려울 때 진면목이 드러난다** — 순풍에는 누구나 좋은 CEO처럼 보인다, 역풍에서 실력이 드러난다
- **자본 배분이 최종 시험이다** — 돈 버는 것은 쉽고, 번 돈을 잘 배분하는 것은 어렵다
- **경영진에게 반하지 마라** — 객관성을 유지하라, 당신이 존경하는 사람도 큰 실수를 저지를 수 있다
