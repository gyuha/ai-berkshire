# ai-berkshire Roadmap

## P0：단기（1-2개월）

### A주 데이터 소스 연동
- akshare, 동방재부(東方財富) 등 무료 데이터 소스 연동
- A주 재무 데이터, 시세, 거래량 상위 종목 커버
- 기존 Skill 수정 불필요, 데이터 레이어 확장만으로 적용 가능

## P1：중기（3-6개월）

### HTML 보고서 출력
- Markdown 기반에 HTML 보고서 포맷 추가
- 다크 모드, 내비게이션 바, 차트 시각화 지원
- 보고서 전파성 및 독서 경험 향상

### 다단계 심층 모드
- `lite`：5분 속판, 빠른 가치 구간 및 핵심 결론 제공
- `standard`：현재 기본 모드, 완전한 멀티 Agent 리서치
- `deep`：교차 검증 및 역사적 유사 사례 추가, 기관급 심층 분석

### 복수 종목 횡단 비교
- 2-4개 종목 동일 기준 횡단 비교 지원
- 동종 업계 기업 가치 벤치마킹
- 비교 매트릭스 및 선택 추천 출력

## P2：장기（6개월+）

### 테스트 커버리지
- 핵심 도구（financial_rigor.py 등）에 단위 테스트 추가
- Skill 출력에 회귀 테스트 추가
- 반복 개발 시 기존 기능 파괴 방지

### 포트폴리오 수준 분석
- 보유 포트폴리오 건전성 평가
- 업종/지역 집중도 분석
- 상관관계 리스크 탐지
