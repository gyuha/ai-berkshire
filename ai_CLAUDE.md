- 투자 스타일: 가치투자, 집중 보유, 중국 인터넷·소비·AI에 집중

- 사용 목적: 개인 투자 의사결정 보조, 동시에 본 프로젝트를 오픈소스 제품으로 홍보

## 프로젝트 핵심 경쟁력 (README에 반영됨)

1. **결론 강제 도출, 양면 눈치 보기 금지** — 통과/불통과/회색지대, 구체적 가격 구간 포함
2. **4대 대가 시각 대결** — 분업이 아니라 서로 도전, 실제 모순과 긴장감 생성
3. **구조적 반편향 메커니즘** — A/B/C 정보 풍부도, 찰리 멍거 역발상, 빠른 거부, 반컨센서스
4. **금융 데이터 정확성** — Decimal 정밀 계산, 시가총액 수기 검산, 다중 출처 교차 검증
5. **재현 가능한 리서치 프로세스** — 동일 입력 → 구조 일관 출력, 횡단 비교·종단 추적 지원
6. **멀티 에이전트 병렬 심화** — 4개 에이전트 각자 검색 + 독립 분석, 4배 정보량
7. **실전 검증** — 2년 누적 146만 수익, 지속적으로 지수 대비 40~50%p 초과수익

## 사용자 선호 및 작업 습관

- **보고서 언어**: 중국어
- **GitHub 푸시**: 리서치 완료 후 보통 푸시 요청, 먼저 확인할 것
- **git 작업**: 원격에 새 커밋이 자주 생김 (사용자가 다른 곳에서도 수정 중일 수 있음), 푸시 전 반드시 `git pull --rebase`
- **오류에 대한 태도**: 직접 지적, 완곡함 불필요. 사용자가 AI 판단에 이의를 제기하면 변명하지 말고 진지하게 재평가할 것
- **과도한 요약 금지**: 사용자는 diff를 볼 수 있으므로, 작업 후마다 한 일을 되풀이할 필요 없음
- **리서치 깊이**: 속도보다 깊이와 정확성 우선, 빠르다고 대충 하지 말 것
