# 한국 기업 버전 데모 README를 별도 파일로 분리

원본 `README.md`는 중국 기업(텐센트·핀둬둬 등)을 한국어로 분석하는 프레임워크 소개 문서다([[0001-reports-in-korean]]: 프로젝트 본류는 "중국 시장 리서치 + 한국어 산출"이며 기업 자체를 바꾸지 않는다). 한국 독자가 친숙한 기업으로 프레임워크 작동 방식을 이해하도록, 중국 기업을 비즈니스 모델이 유사한 한국 상장사로 매핑한 `README_KR_demo.md`를 **별도 파일로 신규 작성**한다. 원본 README·`reports/`·`skills/`는 일절 변경하지 않는다. 이 문서는 소개/데모 자료이며 프로젝트의 리서치 산출물이 아니다.

## Considered Options

- **원본 README를 한국 기업으로 전면 교체**: 트랙레코드(중국 실계좌 +69.29%)·실전 보고서 링크(`reports/腾讯/` 등 실제 디렉터리)·PE 6.3x 같은 실제 분석 데이터가 전부 거짓이 되고, 프로젝트 본류(중국 리서치)가 왜곡됨. 기각.
- **중국어를 한글 표기로만 정리(기업 유지)**: ADR-0001·`CONTEXT.md`의 기존 한글화 궤적 그 자체로, "한국 독자용 친숙한 데모"라는 목적과 다름. 별개 작업으로 분리.
- **한국 기업 데모를 별도 파일 + 판단은 형식 예시로 명시**: 채택.

## Consequences

- `README_KR_demo.md`는 데모/소개용. 실적을 주장하지 않으며, 실제 트랙레코드·보고서는 원본 README를 참조하도록 명시한다.
- 출력 예시의 평점·매수권고·가격구간은 "형식 설명용 예시(실제 투자 권고 아님)"로 표기한다 — 검증 없는 투자 판단을 사실처럼 제시하지 않아 객관성 원칙(`CLAUDE.md`)을 지킨다.
- 팩트 데이터(시총·PER·경영진·주가)는 2026-06-27 기준 실제 교차검증. 단 주가/PER은 시점 의존이라 시간이 지나면 stale — 데모이므로 갱신 의무는 없으며 문서에 기준일을 명시한다.
- 트랙레코드 수익률·중국 계좌 스크린샷·`reports/` 링크는 데모에서 제거한다.
- 1:1 대응이 약한 기업(마오타이·메이퇀·콰이쇼우·팝마트)은 가장 가까운 한국 기업 + 차이점 주석으로 처리한다(하이트진로·쿠팡이츠·SOOP·하이브).
- 4대가(워런 버핏·찰리 멍거·돤융핑·리루)는 프레임워크의 일부이므로 한국 거장으로 바꾸지 않는다.
