# 리포트 산출 언어를 한국어로 전환

기존 `CLAUDE.md`는 "所有报告使用中文"으로 모든 리포트를 중국어로 못 박았다. 프로젝트 전체를 한글화하기로 결정함에 따라, 번역된 스킬이 여전히 "중국어로 작성하라"고 지시하는 모순을 없애기 위해 보고 언어 정책 자체를 **한국어**로 뒤집는다. 이로써 앞으로 스킬이 생성하는 모든 리포트는 한국어로 작성된다.

## Considered Options

- **중국어 유지**: 문자만 한글로 번역하고 정책 문구는 중국어 유지 → 번역된 스킬이 중국어 리포트를 지시하는 모순 발생. 기각.
- **한국어 기본 + 중국어 허용(이중 정책)**: 유연하지만 산출물 언어가 비결정적이 되어 일관성·검증성이 떨어짐. 기각.

## Consequences

- 앞으로 생성되는 리포트는 한국어. 중국 시장(A주·항주) 리서치 도구이지만 산출 언어는 한국어로 고정된다.
- `reports/` 하위 기존 2,083개 중국어 리포트는 이 결정으로 자동 변환되지 않는다 — 별도 리포트 배치 작업(이번 작업의 비목표)에서 처리할지 결정한다.
- 데이터 정확성 원칙(市值手算校验, 데이터 출처 2곳 교차검증 등)은 언어와 무관하게 그대로 유지된다.
