# 한국 기업 버전 데모 README 작성 (README_KR_demo.md)

## Goal / Non-goals
- Goal: 원본 `README.md`를 1:1 구조로 옮긴 한국 기업 버전 데모 문서 `README_KR_demo.md`를 루트에 신규 작성한다. 중국 기업을 비즈니스 모델이 유사한 한국 기업으로 매핑하고, 팩트 데이터(시총·PER·경영진·주가)는 2026-06-27 기준 실제 검증, 투자 판단(평점·권고·가격)은 "형식 예시"로 명시한다.
- Non-goals: 원본 `README.md`·`reports/`·`skills/`·`assets/` 일절 변경 안 함. 실제 한국 기업 투자 분석/권고 생성 안 함(출력 예시는 형식 설명용). 트랙레코드 수익률·중국 계좌 스크린샷·실전 보고서 링크는 제거. 4대가(버핏·멍거·돤융핑·리루)는 한국 거장으로 바꾸지 않음. README_EN.md 한국판은 만들지 않음.

## Source of truth
- Glossary terms: 워런 버핏·찰리 멍거·돤융핑·리루, 경제적 해자, 안전마진, 능력범위, 내재가치, FCF in `.forge/CONTEXT.md` (4대가·핵심 용어 표기는 표준 따름)
- Related ADRs: `.forge/adr/0002-korean-company-demo-readme.md` (이 작업의 근거), `.forge/adr/0001-reports-in-korean.md` (산출 언어 한국어)
- Definition of Done: `README_KR_demo.md`가 루트에 존재하고, 원본 README의 모든 섹션이 한국 기업/용어로 옮겨졌으며, 매핑된 한국 기업의 팩트 데이터가 2개 출처로 검증되었고, 판단성 출력은 "예시"로 명시되었으며, 트랙레코드·스크린샷·`reports/` 링크가 제거되고 원본 참조 안내가 들어가 있다. 원본 `README.md`는 `git diff` 상 변경 없음.

## Work slices
- [ ] S1. 매핑된 한국 상장사(카카오·쿠팡·네이버·크래프톤·하이트진로·SOOP·하이브)의 팩트 데이터(시총·PER·경영진·최근주가, 2026-06-27 기준)를 웹에서 **2개 출처 교차검증**으로 수집. 비상장(토스)·쿠팡이츠는 `[추정]`/주석 처리. — 완료기준: 각 기업 핵심 지표가 출처 2곳 표기와 함께 정리된 데이터 표가 존재
- [ ] S2. 원본 README 구조를 따라 `README_KR_demo.md` 작성: 모든 기업명·예시·산업용어를 매핑대로 교체, 출력 예시 표는 S1 팩트로 채우고 평점·권고·가격엔 "형식 예시" 표기, 4대가 유지, 트랙레코드·스크린샷·`reports/` 링크 제거 후 문서 상단에 "데모용·실적 미주장·기준일 2026-06-27·원본 README 참조" 안내 삽입. — 완료기준: `README_KR_demo.md`가 루트에 존재하고 위 요소가 모두 반영됨 (depends: S1)
- [ ] S3. 검증: 원본 `README.md`가 변경되지 않았는지(`git diff README.md` 빈 결과), 문서 내 중국 기업명/한자 잔존이 없는지(매핑·글로벌·4대가 인용 제외), 판단성 출력에 "예시" 표기가 누락되지 않았는지 점검. — 완료기준: `git diff README.md` 비어 있고, 의도치 않은 잔존 한자·미표기 예시 0건 (depends: S2)
