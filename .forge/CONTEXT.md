# AI Berkshire — 가치투자 리서치

Claude Code 기반 가치투자 리서치 Skill 합집. 4대가 프레임워크(버핏·멍거·돤융핑·리루)의 핵심 용어를, 프로젝트 전반의 중국어→한글 전환에서 **일관되게** 쓰기 위한 표준 용어집이다. 이 용어집은 한글화 작업(skills/, CLAUDE.md 등)과 이후 리포트 배치 작업이 동일한 한국어 표기를 쓰도록 보장한다.

## Language

### 4대가 (고유명)

**워런 버핏**:
프레임워크의 재무·밸류에이션 관점을 대표하는 투자자. 巴菲特.
_Avoid_: 巴菲特, 버펫, 버핏(단독 표기는 본문 맥락에서만)

**찰리 멍거**:
프레임워크의 산업·경쟁·멘탈모델 관점을 대표하는 투자자. 芒格.
_Avoid_: 芒格, 찰리 망거

**돤융핑**:
프레임워크의 비즈니스 모델 관점을 대표하는 투자자. 段永平.
_Avoid_: 段永平, 단영평, 두안용핑

**리루**:
프레임워크의 리스크·경영진 관점을 대표하는 투자자. 李录.
_Avoid_: 李录, 이록, 리 루

### 핵심 투자 용어

**가치투자**:
내재가치 대비 저평가된 자산을 사는 투자 철학. 价值投资.
_Avoid_: 价值投资, 밸류 투자

**경제적 해자**:
경쟁자로부터 기업의 초과수익을 지켜 주는 구조적 우위. 护城河.
_Avoid_: 护城河, 방어막, 진입장벽(해자와 구분)

**안전마진**:
내재가치와 매수가격 사이의 완충 폭. 安全边际.
_Avoid_: 安全边际, 안전 여유

**능력범위**:
투자자가 충분히 이해해 판단할 수 있는 사업 영역의 범위. 能力圈.
_Avoid_: 能力圈, 역량 서클

**내재가치**:
기업이 잔존 수명 동안 창출할 현금흐름의 현재가치. 内在价值.
_Avoid_: 内在价值, 본질가치

**잉여현금흐름 (FCF)**:
영업현금흐름에서 자본적 지출을 차감한 주주 귀속 현금. 自由现金流.
_Avoid_: 自由现金流, 자유현금흐름(FCF로 통일)
